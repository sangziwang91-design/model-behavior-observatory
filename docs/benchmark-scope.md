# Benchmark Scope

## What this benchmark surface tracks

- confidence beyond evidence  
- continuation under incomplete input  
- pseudo-consistency  
- structure drift  
- policy vs execution behavior  

---

## What this does NOT expose

- internal control logic  
- private evaluation pipeline  
- scoring thresholds  
- raw experiment archives  

---

## What counts as a public signal

A public signal is:

- observable  
- repeatable  
- discussable  
- non-reconstructive  

---

## Intended uses

- model comparison  
- report-safe evidence  
- academic citation  
- public understanding  

---

## Boundary

This is a **bounded benchmark surface**, not a full evaluation system.
