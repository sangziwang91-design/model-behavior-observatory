# Known Limits

This repository is about observation, not certainty.

## The central limit

This protocol does **not** measure truth directly.

It measures how a model behaves when truth is **not securely available**.

That distinction matters.

A model can sound coherent, careful, or highly structured while still moving beyond what the evidence securely supports. The protocol here is designed to make that behavior visible. It is not a final truth detector.

## What this protocol can show

- when a model holds a boundary
- when a model expands beyond available evidence
- when a model reframes uncertainty into a plausible-looking structure
- when a model becomes more theory-like under narrative or methodological pressure

## What this protocol cannot claim

- that a model output is globally true or false in a final sense
- that repeated rounds in one session are fully independent trials
- that one protocol run can stand in for all future behavior
- that observed behavior equals internal mechanism

## The self-reference ceiling

There is an unavoidable ceiling here:

the model being observed is also the system producing the observed text.

That means this repository is strongest as a **behavioral observation surface**, not as a perfect external audit layer.

Repeated rounds in a single session are useful for consistency probing, but they are not fully independent trials.

## Why publish this anyway

Because the failure mode is public-facing.

Users encounter confidence signals every day. What they usually do **not** see is where confidence, structure, and reliability come apart.

This repository makes that gap easier to inspect, discuss, and compare.
