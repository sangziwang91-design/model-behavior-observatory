# Zenodo Release: EC-EpC Gap and Behavioral Cartography

Release date: April 29, 2026  
Status: Public Zenodo records available  
Author: SangZi Wang

---

## Release summary

This release establishes two public Zenodo records for pilot-level work on LLM self-report reliability under constrained-access audit conditions.

The release is intentionally bounded. It makes the research problem, public framework, pilot findings, and citation records available without exposing private audit procedures or reverse-engineerable evaluation parameters.

---

## Records

### Record 1 — Preprint

**Auditing Epistemological Credibility in LLM Self-Reports: A Dual-Axis Behavioral Framework and Pilot Evidence**

DOI: [10.5281/zenodo.19879788](https://doi.org/10.5281/zenodo.19879788)

This preprint introduces the EC-EpC gap: the measurable distance between Execution Credibility and Epistemological Credibility in model self-reports. It presents a dual-axis evaluation framework, an eight-class behavioral response taxonomy, and exploratory pilot evidence from a ten-system cross-category study.

### Record 2 — Dataset summary

**Cross-Model Behavioral Cartography: A Pilot Dataset Summary for LLM Self-Report Reliability Research**

DOI: [10.5281/zenodo.19881753](https://doi.org/10.5281/zenodo.19881753)

This dataset summary provides the public empirical companion record for the preprint. It includes session-level structure, cross-model behavioral response pattern comparison, and summary-level quantitative findings.

---

## Claim level

These records are exploratory pilot-level releases.

They support:

- methodological critique
- replication design
- public citation
- discussion of LLM self-report reliability
- bounded external reference

They do **not** support:

- population-level prevalence claims
- model ranking
- causal claims about internal model mechanisms
- validated benchmark status
- regulatory-grade standardization claims

---

## Bounded disclosure

The public records disclose:

- high-level method framing
- behavioral categories
- summary-level quantitative findings
- pilot limitations
- validation requirements

The public records do not disclose:

- exact task inputs
- scoring weights
- trigger thresholds
- intervention templates
- private audit heuristics
- full benchmark task packs
- reverse-engineerable control procedures

This boundary is intentional. It preserves audit integrity while making the research direction citable and externally discussable.

---

## Public framing

The central public question is:

> When a language model explains its own limits, how reliable is that explanation?

The release treats self-description as an observable behavior, not as direct evidence of internal mechanisms.
