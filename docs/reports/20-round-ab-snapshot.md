# 20-Round A/B Snapshot

This page is a **public-safe summary** of a baseline-vs-injection comparison.

It is not a raw export and does not include the full protocol text, internal weighting, or full round-by-round internals.

## Framing

The A/B setup compares:

- **Group A — Baseline**
- **Group B — Structured injection**

The point is not to prove a universal law.
The point is to observe whether structured framing changes how a model handles uncertainty.

## Aggregate result pattern

Public reading of the comparison supports four broad observations:

- fabrication risk rises
- boundary retention falls
- expansion tendency rises sharply
- distortion tendency rises sharply

## Why that matters

The most important movement is usually **not** direct falsehood first.
It is the shift from:

- "this is unverified"

toward:

- "this looks like a plausible framework worth explaining"

In other words, structure can create **premature legitimacy**.

## Public interpretation

This comparison suggests that structured narrative pressure can make a model more willing to:

- organize uncertainty into a theory-like frame
- map unknown topics into familiar concepts
- sound more settled before the truth conditions are secure

## Method warning

Repeated rounds in a single model session are useful for consistency probing, but they are not fully independent trials.

This protocol does **not** measure truth directly.
It measures how a model behaves when truth is not securely available.
