# Core Evidence Entry

This page collects a minimal set of public-safe evidence anchors.

It is designed as a compact entry point for readers who want to see
whether the behavioral findings are grounded in observable patterns.

It does not expose full experimental pipelines, raw data, scoring systems,
or reconstruction-relevant protocol details.

## How to read this page

Each snapshot is intentionally minimal.

Together, they support a broader claim:

> multiple, distinct behavioral patterns
> point to the same mismatch:
> structure and confidence can outrun secure grounding.

## Correct reading boundary

This page is not a reproducibility package.
It is a public evidence layer designed to anchor behavioral claims
without exposing internal system details.
