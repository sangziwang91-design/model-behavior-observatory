# Start Here

This repository is a **public evaluation surface** for model behavior.

If you only read one page first, read this page.

## What this repository is about

The core public question is simple:

> What happens when a model sounds more certain than the evidence securely supports?

This repository does **not** claim to solve that problem.
It makes the mismatch easier to see, compare, and discuss.

## What to keep in mind while reading

Three reading rules matter:

### 1. Observation, not remediation
This repository is for making behavior patterns visible.
It is not a patch, defense, or alignment fix.

### 2. Public-safe, not full disclosure
Some details are intentionally withheld.
This is a boundary decision, not an accidental omission.

### 3. Confidence is not enough
A strong answer can still outrun securely available truth.
That is the central gap this repository is designed to make more visible.

## If you are reading as...

### A general reader
Start with `overview.md`, then go into `findings/index.md`.

### A researcher or journalist
Start with boundaries first, then read the snapshot reports.

### A technical reader
Read findings first, then limits, then the report snapshots.

## What this repository is not

This is **not**:

- a full internal control core
- a leaked private system
- a universal benchmark for all models
- a raw evidence dump

It is a structured public window.

## Best first question to ask while reading

Do not ask only:

- "Is the answer fluent?"

Also ask:

- "Did the model show where secure grounding ends?"
