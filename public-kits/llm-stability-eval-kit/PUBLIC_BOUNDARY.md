# Public Boundary

This package is a public-facing starter kit.

## Safe public concepts

- Repeated-prompt testing
- Output drift observation
- Pseudo-consistency as an output-level phenomenon
- Manual scoring templates
- Lightweight report generation
- RAG / medical / financial example cases

## Not included

- Private model-control protocols
- Proprietary scoring weights
- Seed injection systems
- Hidden thresholds
- Internal audit logic
- Customer-specific reports
- Full research archive

## Rule

This kit should function as an entry point, not as the core system.
